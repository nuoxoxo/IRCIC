[ Subject ] - [intra](https://cdn.intra.42.fr/pdf/pdf/72125/en.subject.pdf) 

- Goal 
  - Create our own **IRC server** 
  - Use an IRC client to to test it ( irssi, nc, etc. )
- Name
  - <kbd> ./ircserve </kbd>
- Run
  - <kbd> ./ircserve _port_  _pass_ </kbd>
- Files allowed
  - Makefile
  - *.hpp, *.cpp, *.tpp, *.ipp
  - *.conf (only one allowed)
